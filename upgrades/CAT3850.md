
## Configuration

**Configure FTP on switch**
```
configure terminal
ip ftp username [uname]
ip ftp password [passwd]
end
```

**Copy image to device**
```
copy ftp: flash:
[ip_address]
[file_name]
```

**Verify MD5 Hash**
```
Switch#verify /md5 flash:[file_name] [expected_md5_hash[Daniel_Arapi_Resume_2021.07.07.docx](https://github.com/darapi/mynetworkstudies/files/6805402/Daniel_Arapi_Resume_2021.07.07.docx)
]
```
